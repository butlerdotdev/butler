# Butler Brand Assets

This directory contains Butler brand assets including logos, diagrams, and illustrations.

## Directory Structure

```
assets/
├── logo/          # Official logos
├── diagrams/      # Architecture diagrams
└── gopher/        # Butler gopher mascot
```

## Logo

The Butler logo is available in multiple formats:

| File | Format | Use Case |
|------|--------|----------|
| `logo/butler-logo.png` | PNG | Web, documentation |
| `logo/butler-logo.svg` | SVG | High-res, print |
| `logo/butler-logo-dark.png` | PNG | Dark backgrounds |
| `logo/butler-icon.png` | PNG | Favicon, small icons |

## Usage Guidelines

### Do

- Use official logo files from this repository
- Maintain aspect ratio when resizing
- Use appropriate version for background color

### Don't

- Modify colors or proportions
- Add effects or shadows
- Use low-resolution versions in print

# Butler v0.1.0 Release Notes

**Release Date**: January 2025

This is the initial alpha release of Butler, providing core multi-cluster management capabilities.

## Highlights

**Management Cluster Bootstrap** - Create Butler management clusters on Harvester or Nutanix infrastructure with a single command.

**Tenant Cluster Provisioning** - Provision tenant Kubernetes clusters with hosted control planes (Kamaji) and worker nodes.

**Addon Management** - Install and manage cluster addons including Cilium, MetalLB, Longhorn, and more.

**Web Console** - Beta web interface for cluster management with real-time updates.

**CLI Tools** - `butleradm` for platform operators and `butlerctl` for platform users.

## Features

### Core Platform

- TenantCluster CRD for declarative cluster management
- Hosted control planes via Kamaji integration
- Team-based multi-tenancy with roles (admin, operator, viewer)
- Single-node and HA topology support

### Providers

- **Harvester HCI**: Full support via CAPI KubeVirt provider
- **Nutanix AHV**: Full support via CAPX

### Addons

Built-in AddonDefinitions for:

- Cilium (CNI) - v1.17.0
- MetalLB (LoadBalancer) - v0.14.9
- Longhorn (Storage) - v1.7.2
- cert-manager (Certificates) - v1.16.2
- Traefik (Ingress) - v34.3.0

### Authentication

- Google SSO (OIDC)
- Local user accounts
- Session-based authentication

### CLI

- `butleradm bootstrap` - Create management clusters
- `butlerctl cluster create/list/get/delete/scale` - Manage clusters
- `butlerctl cluster kubeconfig` - Export kubeconfigs

### Console (Beta)

- Dashboard with cluster overview
- Cluster creation wizard
- Addon management UI
- In-browser terminal
- Real-time WebSocket updates

## Breaking Changes

None (initial release)

## Known Issues

- ManagementAddon Helm operations may fail in some configurations
- Console delete confirmation not implemented (use CLI for safety)
- Apple code signing not available for macOS CLI

## Upgrade Notes

This is the initial release. No upgrade path from previous versions.

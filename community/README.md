# Butler Community

Welcome to the Butler community! This directory contains resources for community participation.

## Getting Involved

### Discussions

- [GitHub Discussions](https://github.com/butlerdotdev/butler/discussions) - Ask questions, share ideas, and connect with other users

### Contributing

- [Contributing Guide](../CONTRIBUTING.md) - How to contribute code, docs, and more
- [Good First Issues](https://github.com/search?q=org%3Abutlerdotdev+label%3A%22good+first+issue%22+state%3Aopen) - Great starting points for new contributors

### Reporting Issues

- Component-specific issues: See [COMPONENTS.md](../COMPONENTS.md) for the right repository
- Cross-cutting issues: Open in [butler/issues](https://github.com/butlerdotdev/butler/issues)

## Meetings

Community meetings are held bi-weekly. See [meetings/](meetings/) for:

- Meeting schedule
- Agenda
- Notes from past meetings
- Recording links (when available)

## How to Get Help

1. **Search existing resources**: Check docs, issues, and discussions
2. **Ask in Discussions**: Post your question with relevant details
3. **Open an Issue**: If you've found a bug or have a feature request

## Stay Updated

- Star the [butler repository](https://github.com/butlerdotdev/butler)
- Watch for releases
- Follow release announcements in Discussions
